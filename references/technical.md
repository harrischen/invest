# 技术面分析模块

**技术面是战术工具——回答"怎么打赢"这个问题。多数时候，我们赚的都是趋势的钱。**

## 数据获取现实约束

> **重要**：技术面分析依赖实时行情数据，但当前工具能力存在以下限制：
> - 财经网站反爬机制可能导致 agent-browser 抓取失败
> - WebSearch 返回的是摘要片段，不一定包含精确的均线/成交量数值
> - 数据可能有滞后（非实时）
>
> **处理原则**：
> - 能获取精确数据时 → 使用量化指标判断
> - 只能获取部分数据时 → 结合可得数据 + 定性逻辑判断趋势阶段
> - 完全无法获取时 → 标注 N/A，基于 52 周高低点、近期新闻情绪等做保守判断，并标注置信度

### 数据获取步骤

```
Step 1: 获取当前股价和近期走势
  → agent-browser 访问行情页面
  → 提取：当前价、涨跌幅、5/10/20/60日均线值、成交量

Step 2: 计算趋势指标
  → 均线排列关系（多头/空头/缠绕）
  → 近20日涨跌幅、近60日涨跌幅
  → 价格相对60日均线的偏离度

Step 3: 获取筹码分布（A股）
  → agent-browser 访问同花顺筹码页面
  → 识别主要筹码峰位置和集中度

Step 4: 获取成交量和资金流向
  → 近5日 vs 近20日平均成交量对比（放量/缩量）
  → 主力资金净流入/流出

Step 5: 综合判断趋势阶段
```

## 趋势判断体系

**核心思路**：用可量化的指标来判断趋势阶段，而非依赖视觉判断。

### 趋势五阶段模型

| 阶段 | 定义 | 核心特征 | 操作 |
|------|------|----------|------|
| **强势上行** | 趋势确认且加速 | 均线多头 + 价格远高于60日线 + 放量 | ⚠️ 警惕过热，不追高 |
| **健康上行** | 稳健上升趋势 | 均线多头 + 价格在60日线上方适度距离 + 量价配合 | ✅ **最佳持有/买入** |
| **震荡整理** | 方向不明 | 均线走平缠绕 + 价格围绕均线波动 + 缩量 | ⏸️ 观望，等方向 |
| **弱势下行** | 下跌趋势确认 | 均线空头 + 价格低于60日线 + 反弹无力 | ❌ 不做多，止损 |
| **超跌筑底** | 跌无可跌，底部区域 | 大幅低于60日线 + 极度缩量 + 利空不跌 | 👀 关注转向信号，不急 |

### 量化判断标准

以下指标用于客观判断当前处于哪个阶段：

#### 指标1：均线排列

| 状态 | 判断标准 | 含义 |
|------|----------|------|
| 多头排列 | 5日 > 10日 > 20日 > 60日均线 | 趋势向上 |
| 空头排列 | 5日 < 10日 < 20日 < 60日均线 | 趋势向下 |
| 缠绕/交叉 | 均线频繁交叉，无明确方向 | 震荡/方向不明 |

#### 指标2：价格偏离度（相对60日均线）

| 偏离度 | 计算公式 | 含义 |
|--------|----------|------|
| > +20% | (现价-MA60)/MA60 | 严重超涨，过热 |
| +5% ~ +20% | | 健康上行区间 |
| -5% ~ +5% | | 围绕均线震荡 |
| -5% ~ -20% | | 弱势下行 |
| < -20% | | 严重超跌 |

#### 指标3：动量（近期涨跌幅）

| 周期 | 强势 | 健康 | 中性 | 弱势 | 极弱 |
|------|------|------|------|------|------|
| 近20日 | >15% | 5~15% | -5~5% | -15~-5% | <-15% |
| 近60日 | >30% | 10~30% | -10~10% | -30~-10% | <-30% |

#### 指标4：成交量状态

| 状态 | 判断标准 | 含义 |
|------|----------|------|
| 放量 | 近5日均量 > 近20日均量 × 1.5 | 资金活跃，关注方向 |
| 正常 | 近5日均量 ≈ 近20日均量（0.7~1.5倍） | 正常交投 |
| 缩量 | 近5日均量 < 近20日均量 × 0.7 | 观望情绪重 |

#### 指标5：MACD状态

| 状态 | 含义 |
|------|------|
| DIF > 0 且金叉 | 趋势向上确认 |
| DIF > 0 且死叉 | 上升途中调整 |
| DIF < 0 且金叉 | 可能触底反转 |
| DIF < 0 且死叉 | 下跌趋势延续 |

### 综合判断矩阵

| 阶段 | 均线 | 偏离度 | 20日动量 | 成交量 | MACD |
|------|------|--------|----------|--------|------|
| 强势上行 | 多头 | > +15% | > +10% | 放量 | DIF>0 金叉 |
| **健康上行** | **多头** | **+5~+15%** | **+3~+10%** | **正常/温和放量** | **DIF>0** |
| 震荡整理 | 缠绕 | -5~+5% | -5~+5% | 缩量 | DIF≈0 |
| 弱势下行 | 空头 | -5~-20% | -5~-15% | 正常/放量 | DIF<0 死叉 |
| 超跌筑底 | 空头 | < -20% | < -15% | 极度缩量 | DIF<0 但收敛 |

**判断原则**：不要求所有指标完全吻合，多数指标指向同一阶段即可判定。

## 筹码峰与公允价格

**筹码峰 = 公允价格 = 历史上多空双方认可的价格区域**

### 识别方法
- 过去6个月~1年成交量最大的价格区间
- 60日均线附近（中期筹码密集区）
- 多根均线汇聚处
- 前期平台整理区间（横盘时间越长、成交量越大 = 筹码峰越强）

### 量化识别标准
- **主力筹码峰**：区间内累计成交占总成交 > 30%
- **次要筹码峰**：区间内累计成交占总成交 15~30%
- **峰的宽度**：通常为股价的 ±5~10% 范围

### 作用
- **下方筹码峰 = 支撑**（持有者成本在此，不愿亏损卖出）
- **上方筹码峰 = 压力**（套牢盘解套会卖出）
- **回踩筹码峰 = 买入机会**（安全边际高，止损明确）

### 支撑/压力强度判断

| 强度 | 条件 |
|------|------|
| 强支撑/压力 | 筹码集中度高(>30%) + 持续时间长(>3个月) + 成交量大 |
| 中等 | 筹码集中度中(15-30%) + 持续1-3个月 |
| 弱 | 筹码集中度低(<15%) + 持续时间短 |

## 买入三条件

```
1. ✓ 商业模式靠谱（基本面OK）
2. ✓ 趋势处于"健康上行"阶段（均线多头 + 偏离度适中 + 动量正向）
3. ✓ 股价回撤至重要筹码峰附近（回调到支撑位）

前提：基本面与趋势方向一致（都向上）
```

### 最佳买入时机的量化条件

| 条件 | 量化标准 |
|------|----------|
| 趋势确认 | 均线多头排列，MACD DIF > 0 |
| 回调幅度合适 | 从近期高点回撤 5~15%（非深跌） |
| 到达支撑位 | 接近筹码峰 / 20日或60日均线 |
| 缩量回调 | 回调时成交量萎缩（非放量下跌） |
| 企稳信号 | 出现止跌K线（下影线、小阳线） |

## 风险管理

### 核心规则
1. **趋势优先**：当趋势指标与你的主观预期矛盾时，尊重趋势指标
2. **计划先行**：买入前明确止损位、目标位、仓位
3. **依计划行事**：不因情绪违反计划

### 仓位管理
- 单只 ≤ 20-30%，高确信不超过40%
- 新建仓先轻仓试探（10-15%），确认后加仓
- 只在盈利时加仓，不在亏损时补仓

### 止损
- 个股：-7% ~ -10%
- 跌破筹码峰下沿 = 离场
- 均线从多头转为空头 = 离场信号
- 总组合回撤：-15%（触发后降低整体仓位）

### 盈利保护
- 盈利 > 15%：止损上移至成本价
- 盈利 > 30%：止损上移至 +15%
- 分批止盈：1/3 → 1/3 → 剩余让利润奔跑
