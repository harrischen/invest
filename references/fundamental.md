# 基本面分析模块

**基本面是定性工具——回答"值不值得参与、打不打得赢"这个战略问题。基本面不可以直接用来做交易。**

### 数据获取步骤

分析一家公司时，按以下顺序获取数据：

```
Step 1: 搜索公司基本信息
  → WebSearch: "[公司名] 主营业务 商业模式 竞争格局"

Step 2: 获取财务数据（最近3年）
  → A股: agent-browser 访问东方财富财务分析页面
  → 港股: agent-browser 访问雪球/富途财务页面
  → 提取: 营收、净利润、EBIT、经营现金流、自由现金流

Step 3: 获取估值数据
  → WebSearch: "[公司名] PE PB PS 估值 历史分位"
  → 或 agent-browser 抓取同花顺/雪球估值页面

Step 4: 获取行业对比数据
  → WebSearch: "[公司名] 同行业 对比 市占率"
  → WebSearch: "[行业] 竞争格局 龙头"

Step 5: 补充信息（可选）
  → 最新研报观点: WebSearch "[公司名] 研报 2025"
  → 最新公告: 巨潮/港交所
```

## 三个维度

### 1. 商业模式

**核心问题**：
- 做什么生意？赚谁的钱？
- 有没有护城河？什么类型？
- 能否规模化扩张？天花板在哪？

**护城河类型**：

| 类型 | 判断标准 | 典型案例 |
|------|----------|----------|
| 品牌效应 | 消费者愿为品牌溢价付费 | 茅台、苹果 |
| 网络效应 | 用户越多产品越有价值 | 微信、淘宝 |
| 成本优势 | 规模/资源带来低成本 | 宁德时代、海螺水泥 |
| 转换成本 | 客户换供应商代价高 | Oracle、SAP |
| 牌照/特许 | 政策壁垒 | 银行、免税 |
| 技术壁垒 | 核心技术难复制 | 台积电、ASML |

**伟大公司四条件**（每项评 ✓/✗/△）：
1. **无差别产品**：标准化，不依赖个人，可大规模复制
2. **最广泛用户**：潜在客户群大，全年龄/全地域
3. **自由定价**：提价不流失客户，有定价权
4. **市场垄断**：市占率领先(>30%)，新进入者难

---

### 2. 财务表现

**达标标准**：

| 指标 | 合格 | 优秀 |
|------|------|------|
| 年收入 | > 1亿美元（~7亿RMB） | > 10亿美元 |
| 收入年增速 | 15-20% | > 30% |
| EBIT Margin | 行业中上 | 行业Top 20% |
| 经营现金流 | > 0 | > 净利润 |
| 自由现金流 | > 0 | 连续3年为正 |

**财务质量红旗**：
- 应收增速 > 营收增速 → ⚠️ 赊销放宽
- 存货增速 > 营收增速 → ⚠️ 积压
- 经营现金流/净利润 < 0.7 → ⚠️ 利润注水
- 商誉/净资产 > 30% → ⚠️ 减值风险
- 短期借款 > 现金 → ⚠️ 流动性压力

---

### 3. 市场定价

**估值指标**：

| 指标 | 适用场景 | 便宜 | 合理 | 贵 |
|------|----------|------|------|-----|
| P/E | 盈利稳定公司 | <15 | 15-25 | >40 |
| P/S | 高增长未盈利 | <3 | 3-10 | >10 |
| P/B | 重资产行业 | <1 | 1-3 | >5 |
| EV/EBITDA | 跨国对比 | <8 | 8-15 | >15 |

**评估方法**：
1. 历史分位：当前 vs 过去5年（<20%=低估，>80%=高估）
2. 同行对比：vs 可比公司，分析溢价/折价原因
3. 隐含预期：当前P/E隐含未来需要多高增速才合理？
