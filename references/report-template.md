# 研报输出模板

## 完整研报格式

```markdown
## [公司名称]（[代码]）投资分析报告

**日期**：[YYYY-MM-DD] | **股价**：[xxx] | **市值**：[xxx亿]

---

### 结论

> **[强烈推荐/推荐/中性/谨慎]** — [一句话核心理由]

### 核心逻辑（≤3点）
1. [逻辑1]
2. [逻辑2]
3. [逻辑3]

---

### 一、产业定位
- **所在产业**：[xxx]
- **产业周期**：[上行/顶部/下行/底部]
- **公司位置**：[上游/中游/下游] — [具体环节]
- **是否受益环节**：[是/否] — [原因]
- **产业景气度**：[x]/5

---

### 二、基本面

#### 商业模式
| 条件 | 评价 | 说明 |
|------|------|------|
| 无差别产品 | ✓/✗/△ | ... |
| 最广泛用户 | ✓/✗/△ | ... |
| 自由定价 | ✓/✗/△ | ... |
| 市场垄断 | ✓/✗/△ | ... |

护城河：[类型]

#### 财务表现
| 指标 | 2022 | 2023 | 2024 | 达标 |
|------|------|------|------|------|
| 营收(亿) | | | | |
| 营收增速 | | | | ✓/✗ |
| EBIT Margin | | | | ✓/✗ |
| 经营现金流 | | | | ✓/✗ |
| 自由现金流 | | | | ✓/✗ |

#### 市场定价
| 指标 | 当前 | 历史分位 | 同行 | 判断 |
|------|------|----------|------|------|
| P/E | | | | |
| P/S | | | | |

**公司质地**：[x]/5

---

### 三、技术面
- **趋势阶段**：[强势上行/健康上行/震荡整理/弱势下行/超跌筑底]
- **判断依据**：[1-2句话说明基于什么数据/信息得出此判断]
- **关键位置**：
  - 当前价 [xxx] / 支撑位 [xxx]（-x%）/ 压力位 [xxx]（+x%）
- **一致性**：基本面[↑/→/↓] vs 图形[↑/→/↓] = [一致✓/不一致⚠️]
- **数据置信度**：[高/中/低]（高=实时数据，中=搜索摘要，低=信息有限）
- **技术位置**：[x]/5 — [1句话依据]

---

### 四、综合评分

| 维度 | 权重 | 评分 | 加权 | 依据 |
|------|------|------|------|------|
| 产业景气度 | 30% | [x]/5 | [x] | [1句话] |
| 公司质地 | 40% | [x]/5 | [x] | [1句话] |
| 技术位置 | 30% | [x]/5 | [x] | [1句话] |
| **总分** | | | **[x.x]/5** | |

评级：4.0-5.0 强烈推荐 | 3.0-3.9 推荐 | 2.0-2.9 中性 | <2.0 谨慎

---

### 五、操作建议

| 项目 | 建议 |
|------|------|
| 方向 | [看多/观望/回避] |
| 买入区间 | [xxx ~ xxx] |
| 止损 | [xxx]（-x%） |
| 目标1 | [xxx]（+x%） |
| 目标2 | [xxx]（+x%） |
| 仓位 | 总资金 x% |

---

### 六、风险提示
1. [xxx]
2. [xxx]
3. [xxx]

---

### 七、数据来源与局限性

- **数据来源**：[逐条列出关键数据的来源]
- **数据时效**：[标注数据获取日期，说明可能的滞后]
- **已知局限**：[说明哪些数据未能获取、哪些判断基于有限信息]

---

> 仅供参考，不构成投资建议。投资有风险，入市需谨慎。
```

## 快速摘要格式

```markdown
## [公司]（[代码]）| 快速分析

**日期**：[YYYY-MM-DD] | **股价**：[xxx]

> **[推荐/中性/谨慎]** — [一句话]

| | 数值 | 判断 |
|---|---|---|
| 营收增速 | xx% | ✓/✗ |
| EBIT Margin | xx% | ✓/✗ |
| FCF | xx亿 | ✓/✗ |
| P/E(分位) | xx(xx%) | 贵/合理/便宜 |
| 时钟 | x点 | [描述] |

**评分**：产业[x] × 质地[x] × 位置[x] = **[x.x]/5**

**操作**：买入[xxx~xxx] | 止损[xxx] | 目标[xxx]
```
