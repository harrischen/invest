## 快速摘要格式

```markdown
## [公司]（[代码]）| 快速分析

**日期**：[YYYY-MM-DD] | **股价**：[xxx]

> **[推荐/中性/谨慎]** — [一句话]

| | 数值 | 判断 |
|---|---|---|
| 营收增速 | xx% | ✓/✗ |
| EBIT Margin | xx% | ✓/✗ |
| FCF | xx亿 | ✓/✗ |
| P/E(分位) | xx(xx%) | 贵/合理/便宜 |
| 技术面总分 | x.x/5 | [阶段] |

**评分**：产业[x]×30% + 质地[x]×40% + 技术[x]×30% = **[x.x]/5**

**操作**：买入[xxx~xxx] | 止损[xxx] | 目标[xxx]
```
