# 风险管理规则

**有预期、有底线、有计划，最重要的是：依计划行事。**

## 规则一：图形优先

> 图形与预期不一致时，**必须尊重图形方向**。

- 基本面好但股价跌 → 尊重图形，先离场
- 你觉得该涨但图形是4点钟 → 尊重图形，不做多
- 利好出来不涨反跌 → 市场在告诉你不知道的事

**原因**：图形反映了所有已知+未知信息的综合。如果基本面好但价格跌，说明有你不知道的信息在起作用。

## 规则二：计划先行

买入前必须明确：

| 问题 | 必须回答 |
|------|----------|
| 为什么买？ | 逻辑 |
| 买入价？ | 具体价格区间 |
| 买多少？ | 仓位比例 |
| 目标价？ | 盈利目标 |
| 止损价？ | 最大亏损 |
| 持有多久？ | 时间预期 |

## 规则三：仓位管理

- 单只 ≤ 20-30%（高确信不超过40%）
- 新建仓：先轻仓10-15%，确认后加仓
- **只在盈利时加仓**（趋势确认后加）
- **绝不在亏损时补仓**

## 规则四：止损

- 个股止损：-7% ~ -10%
- 总组合回撤止损：-15%（触发后降低整体仓位）
- 跌破筹码峰下沿 = 离场
- 止损后不立即追回，等待新信号

## 规则五：盈利保护

- 盈利 > 15%：止损上移至成本价（绝不让盈利变亏损）
- 盈利 > 30%：止损上移至 +15%
- 分批止盈：到目标1减1/3 → 到目标2减1/3 → 剩余让利润奔跑

## 规则六：情绪管理

**危险信号**：
- 亏损后急于翻本 → 容易重仓博弈
- 赚钱后飘了 → 忽视风险
- 频繁看盘 → 被短期波动干扰
- 不确定时的最佳操作是：**不操作**

## 输出格式

```markdown
### 风险管理方案

| 项目 | 数值 |
|------|------|
| 建议仓位 | 总资金 x% |
| 买入区间 | xxx ~ xxx |
| 止损价 | xxx（-x%） |
| 第一目标 | xxx（+x%） |
| 第二目标 | xxx（+x%） |
| 移动止损 | 盈利>15%后上移至成本 |

### 情景预案
- 买入后继续跌至止损 → 止损离场，不补仓
- 买入后快速上涨 → 不追加，等回踩加仓
- 长期横盘不动 → 设时间止损（如3个月无变化则退出）
- 出现重大利空 → 评估逻辑是否被破坏，若是则立即离场
```
