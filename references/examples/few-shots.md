## 示例二：技术面分析 — 数据缺失版（L3 降级）

**输入**：分析比亚迪（002594.SZ）技术面

**输出**：

```markdown

### 趋势判断（定性）
- **趋势阶段**：震荡整理 — 判断依据：近3个月股价在 280~320 区间波动，无明确方向突破 [搜索摘要, 2025-06]

- **近期表现**：过去20日涨幅约 +3%，整体平稳
- **市场情绪**：多数机构维持"增持"评级，但短期缺乏催化剂 [搜索摘要]

### 位置分析（简化）

- **相对52周区间位置**：中偏上（约 62% 分位）
- **可参考支撑**：280 元（近期多次回踩不破）
- **可参考压力**：320 元（近期多次冲高回落）

- 价格趋势方向：→（横盘震荡）
- **一致性**：不一致 ⚠️ — 基本面向好但图形横盘，尊重图形，等方向确认

- [ ] 近期成交量是否放大
```

## 示例三：趋势雷达 — 单条强信号

**输入**：扫描最近的投资方向

```markdown

- **核心证据**：

  3. 工信部发布《算力基础设施发展行动计划》[gov.cn, 2025-04]

- **A股/港股方向**：中际旭创、新易盛（光模块）；英维克（液冷）；沪电股份（PCB）
- **下一步**：拆解 AI 算力产业链，找定价不充分的环节
```

```markdown
| 技术位置 | 30% | 3.0/5 | 0.90 | 震荡整理中，方向未确认，数据置信度中 |
| **总分** | | | **3.50/5** | |

计算：1.20 + 1.40 + 0.90 = 3.50

评级：**推荐**（3.0-3.9区间）
```

## 示例五：数据获取失败的标准处理

当某项数据完全无法获取时的写法：

```markdown
#### 财务表现
| 指标 | 数值 | 达标 |
|------|------|------|
| 年收入 | 2,450亿 [财报, 2024] | ✓ |
| 收入增速 | 18% [财报, 2024] | ✓ |
| EBIT Margin | N/A（获取失败：页面未加载） | — |
| 经营现金流 | 正 [搜索摘要, 精确值未获取] | ✓ |
| 自由现金流 | N/A（获取失败：数据源无此字段） | — |

> 注：EBIT Margin 和自由现金流数据未能获取，公司质地评分取保守值 3.0/5（若数据完整可能更高）。
```
