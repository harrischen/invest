# 产业链分析模块

## 核心逻辑

**大趋势确认后 → 拆解产业链 → 找到瓶颈/增量环节 → 验证供需 → 筛选标的**

## 拆解方法

问三个问题：
1. 这东西要落地，**物理上需要什么？**（硬件/设备/材料）
2. 这东西要运转，**逻辑上需要什么？**（软件/服务/运营）
3. 这东西要扩张，**瓶颈在哪里？**（供给最紧的环节）

## 标准产业链结构

```
上游（原材料/设备/零部件）→ 中游（制造/集成）→ 下游（应用/客户）→ 配套（基础设施）
```

## 找到最佳环节

### 瓶颈型（供给跟不上需求）

| 信号 | 含义 |
|------|------|
| 产品涨价 | 供不应求 |
| 交期拉长 | 订单排满 |
| 产能利用率>90% | 满产 |
| 扩产周期长（>2年） | 短期供给无法增加 |
| 只有少数玩家能做 | 技术壁垒 |

### 增量型（用量暴增）

| 信号 | 含义 |
|------|------|
| 单位用量提升 | 新标准/新技术带来更多需求 |
| 渗透率加速 | 从低到高的S曲线拐点 |
| 国产替代 | 份额从0到N |
| 下游爆发 | 终端市场快速扩大 |

### 环节评分（1-5分）

| 维度 | 1分 | 3分 | 5分 |
|------|-----|-----|-----|
| 需求增速 | <10% | 20-30% | >50% |
| 供给约束 | 易扩产 | 1年建设期 | >2年+技术壁垒 |
| 竞争格局 | 极分散 | 3-5家 | 寡头垄断 |
| 涨价能力 | 持续降价 | 价格稳定 | 主动涨价 |
| 确定性 | 远期预期 | 明年兑现 | 当前已在兑现 |

**总分 > 18：重点关注 | 13-18：值得跟踪 | < 13：优先级低**

## 供需验证

```
需求端: "[下游] 资本开支 扩产 2025 2026"
供给端: "[产品] 全球产能 在建产能"
价格端: "[产品] 价格走势 涨价 2025"
订单端: "[龙头公司] 订单 产能利用率"
```
